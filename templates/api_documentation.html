API Документация

Возможности API

  • Получение расписания

Для получения расписание, необходимо выполнить POST запрос

И передать Json список с данными аргументами

  • В 'name_klass' передать название класса
  • В 'shool_id' передать id школы в котором находится класс
  • В 'date' передать дату на которую необходимо расписание в формате 'год-месяц-день'
  • Так же в 'data' можно передать в место даты 'all', тогда будет возвращено всё имеещееся расписание

Пример запроса на Python

from requests import post 

print(post('http://localhost:5000/api/sheduele',
           json={'name_klass': '1б', 
                 'shool_id': 2, 
                 'date': '2021-04-16'}).json())
